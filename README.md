# Engram — Presentation Site

Landing page for the [Engram](https://github.com/dominikj111/Engram) project.

Built with React + TypeScript + Vite. Deployed at **[dominikj111.github.io/engram](https://dominikj111.github.io/engram/)**.

## Run locally

```bash
npm install
npm run dev
```

## About Engram

Engram is a deterministic reasoning kernel — a confidence-weighted finite state machine
that navigates a directed graph of concepts, resolves ambiguity through predefined context
hooks and breaking questions, and emits typed action contracts for a separate execution layer.

→ [github.com/dominikj111/Engram](https://github.com/dominikj111/Engram)
